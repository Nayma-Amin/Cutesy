# Cutesy - Shopping App

Cutesy is a flutter and dart mobile application. A shopping app, designed with simplicity, implemented with unique and dynamic solutions. From the flash screen till the backend it is implemented to enhence user engagement and behold user interest.

## Features

- User sign up and login
- Users interact with products, cart, orders and reviews.
- All the product, discounts, reviews, users and coupons. 
- User profile to edit profile and manage informations
- Users add products to cart to order later
- User can add products to favourites
- Admin dashboard maintains authenticity
